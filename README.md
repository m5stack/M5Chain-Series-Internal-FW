# STM32 Internal Firmware Repository for M5Stack Chain Series

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![M5Stack](https://img.shields.io/badge/M5Stack-Chain%20Series-orange.svg)](https://docs.m5stack.com)

This repository hosts the STM32 internal firmware for the M5Stack Chain series devices, providing core functionality for various sensor and control modules.

## 📚 Related Repositories

- **Arduino Library**: [m5stack/M5Chain](https://github.com/m5stack/M5Chain) - Official Arduino library for Chain devices
- **Chain ID Management**: [m5stack/m5stack-chain-id](https://github.com/m5stack/m5stack-chain-id) - Device identification and management tools
